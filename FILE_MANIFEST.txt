   → Follow USAGE_GUIDE.md

================================================================================
IMPORTANT NOTES
================================================================================

1. LARGE FILES
   - .gitignore already excludes large data files
   - OK to commit: scripts, summaries, CSVs, small structures
   - NOT OK: Raw .pdb files (download from RCSB), large .dlg files

2. PYTHON ENVIRONMENT
   - Create venv: python -m venv venv
   - Activate: source venv/bin/activate (Mac/Linux)
   - Install: pip install -r requirements.txt

3. GITHUB AUTHENTICATION
   - Use SSH key (recommended) or Personal Access Token
   - See GITHUB_SETUP.md section 2

4. COLLABORATION
   - Make repository PUBLIC for research sharing
   - Add CONTRIBUTING.md if inviting contributors
   - Use Issues for tracking tasks/bugs

5. BACKING UP RESULTS
   - GitHub provides version control (automatic backup)
   - For extra safety: periodic local backups
   - Consider Zenodo/OSF for long-term archival
